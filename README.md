<H3>Command UART</H3>
<p>MXY X_VAL Y_VAL</p>
<p>Move pen to X-Axis and Y-Axis positions </p>
<p>PEN P_VAL</p>
<p>P_VAL = 1 => pen up. P_VAL = 0 => pen down.</p>
